## Entidades del Sistema

Para el desarrollo de este proyecto se han definido un total de **15 entidades** (tablas)

1. **empresa**: Almacena los datos de las empresas de transporte.
2. **ruta**: Define las rutas asignadas a las empresas.
3. **unidad**: Registra los vehículos (buses/combis) operativos.
4. **chofer**: Contiene la información de los conductores de las unidades.
5. **pasajero**: Almacena los datos personales de los usuarios del sistema.
6. **tarjeta**: Registra las tarjetas de transporte vinculadas a los pasajeros.
7. **tarifa**: Define los costos de los pasajes según el tipo de viaje.
8. **transaccion**: Registra cada pago o recarga realizada con la tarjeta.
9. **reporte_ingresos**: Consolida los ingresos económicos por tarifas y empresas.
10. **tipo_usuario**: Clasifica los beneficios y descuentos (Universitario, Escolar, etc.).
11. **incidencia**: Registra reportes de problemas, pérdidas o fallas con las tarjetas.
12. **tipo_tarjeta**: Especifica la modalidad del servicio (Prepago o Postpago).
13. **bloqueo_tarjeta**: Lleva el control de las tarjetas suspendidas y sus motivos.
14. **notificacion_pasajero**: Almacena las alertas enviadas a los usuarios sobre su estado.
15. **historial_cambio_tarjeta**: Registra las renovaciones o sustituciones físicas de las tarjetas.
